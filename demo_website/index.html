Libraries we used

  • pandas, a python data analysis library
  • Keras, a python deep learning library
  • scikit-learn, a python machine learning library
  • sklearn-porter, to transpile sklearn models to JS

Dataset description

This dataset comes from the "Sentiment Analysis on Movie Reviews" Kaggle Challenge. It was extracted from the Rotten Tomatoes Website.
